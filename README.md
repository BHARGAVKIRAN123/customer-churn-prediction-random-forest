# Customer Churn Prediction using Random Forest

This project demonstrates a Machine Learning model that predicts whether a customer will leave (churn) or stay using Random Forest.

## 📌 Problem Statement
Predict customer churn based on features like age, monthly charges, tenure, and contract type.

## 🔍 Concepts Used
- Data preprocessing
- Categorical encoding
- Train-Test Split
- Random Forest Classifier
- Model prediction and evaluation
- Feature importance analysis

## 🛠 Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## 📊 Dataset
Custom dataset with customer details:
- Age
- Monthly Charges
- Tenure
- Contract Type
- Churn (Yes/No)

## ⚙️ Workflow
1. Load dataset
2. Convert categorical data to numeric
3. Split into training and testing sets
4. Train Random Forest model
5. Predict churn
6. Evaluate accuracy
7. Analyze feature importance

## 📈 Result
Achieved good accuracy in predicting customer churn.

## 💡 Key Learning
- Understood how Random Forest improves prediction accuracy
- Learned how to handle categorical data
- Practiced end-to-end machine learning workflow
